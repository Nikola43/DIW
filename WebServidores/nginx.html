1. Instalación Servidor nginx

Instalación

Debemos abrir el terminal y ejecutar el siguiente comando:

$ sudo apt install nginx -y

Al terminar, comprobamos que se ha instalado correctamente con el siguiente comando:

$ nginx -v

Debemos obtener un resultado como este:

Ahora iniciamos el servidor con el siguiente comando:

$ sudo service nginx start

Comprobamos si el servidor esta funcionando con el siguiente comando:

$ sudo service nginx status

2. Configurando el servidor

Permitir ejecución de ficheros .php

Ahora mismo si intentamos entrar en la web del servidor, nos dará como respuesta '403 Forbbiden' ya que el servidor no permite la ejecución de ficheros .php. Para habilitar la ejecución de codigo php, debemos editar el fichero /etc/nginx/sites-enabled/default

Para editar el fichero usamos el siguiente comando:

$ sudo vim /etc/nginx/sites-enabled/default

Debemos añadir index.php al final de la lista de ficheros admitidos.

Guardamos el fichero y reiniciamos el servidor con el siguiente comando:

$ sudo service nginx restart

3. Probando el servidor

Prueba en el navegador

Abrimos el navegador y vamos a la dirección http://localhost/. Si todo ha ido bien, debería mostrar la web.

Como se puede observar, el servidor genera una web html mostrando la version de php instalada.
